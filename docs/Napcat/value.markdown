## 全局变量
| 变量名 | 介绍 |
|-------|-------|
| raw | 原始元数据（JSON数组） |
| 登录账号 | 机器人QQ号（从config中获取） |
| 消息来源 | 群聊<br>私聊<br>群聊-入群<br>群聊-退群<br>群聊-消息撤回<br>私聊-消息撤回<br>群聊-群成员禁言<br>群聊-群成员解禁<br>群聊-管理员添加<br>群聊-管理员减少<br>群聊-申请入群 |

## 群聊
| 变量名 | 介绍 |
|-------|-------|
| raw | 原始元数据 |
| 消息 | 经过CQ_message处理后的用户消息 |
| 群号 | 消息来源的群号 |
| 消息ID | 消息的唯一标识符 |
| QQ | 发送消息的用户QQ号 |
| 昵称 | 发送消息的用户昵称 |
| 身份 | 群主<br>管理员<br>群员 |
| 群名 | 群聊名称 |

## 私聊
| 变量名 | 介绍 |
|-------|-------|
| raw | 原始元数据 |
| 消息 | 经过CQ_message处理后的用户消息 |
| 消息ID | 消息的唯一标识符 |
| QQ | 发送消息的用户QQ号 |
| 昵称 | 发送消息的用户昵称 |

## 通知事件
| 事件类型 | 变量名 | 介绍 |
|---------|-------|------|
| **群聊-入群** | QQ | 入群用户的QQ号 |
| | 群号 | 入群的群号 |
| **群聊-退群** | QQ | 退群用户的QQ号 |
| | 群号 | 退群的群号 |
| **群聊-消息撤回** | QQ | 撤回消息的用户QQ号 |
| | 群号 | 发生撤回的群号 |
| | 消息ID | 被撤回消息的ID |
| **私聊-消息撤回** | QQ | 撤回消息的用户QQ号 |
| | 消息ID | 被撤回消息的ID |
| **群聊-群成员禁言** | QQ | 被禁言的用户QQ号 |
| | 群号 | 发生禁言的群号 |
| | 操作者 | 执行禁言操作的管理员QQ号 |
| | 禁言时长 | 禁言时长（秒） |
| **群聊-群成员解禁** | QQ | 被解禁的用户QQ号 |
| | 群号 | 发生解禁的群号 |
| | 操作者 | 执行解禁操作的管理员QQ号 |
| **群聊-管理员添加** | QQ | 被添加为管理员的用户QQ号 |
| | 群号 | 发生添加的群号 |
| **群聊-管理员减少** | QQ | 被移除管理员的用户QQ号 |
| | 群号 | 发生移除的群号 |
| **群聊-申请入群** | QQ | 申请入群的用户QQ号 |
| | 群号 | 申请加入的群号 |
| | 申请ID | 申请的唯一标识符（用于处理申请） |

## 特殊说明
所有变量都是通过`define()`定义的常量，在插件中可直接使用
