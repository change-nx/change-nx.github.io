## 函数用法

### 消息构造

#### `text($content)`
纯文本消息段。`$content` — 文本内容
```php
群(群号, text("你好"));
```

#### `reply($content)`
回复消息段。`$content` — 被回复的消息 ID
```php
群(群号, reply(消息ID), text("收到"));
```

#### `face($id)`
QQ 表情。`$id` — 表情 ID
```php
群(群号, face(21));
```

#### `image($content)`
图片消息段。`$content` — 图片路径/URL/base64
```php
群(群号, image("https://example.com/1.png"));
```

#### `video($content)`
视频消息段。`$content` — 视频路径/URL
```php
群(群号, video("https://example.com/1.mp4"));
```

#### `music($type, $id)`
音乐卡片。`$type` — 平台（`qq`/`163`）；`$id` — 歌曲 ID
```php
群(群号, music("163", "1234567"));
```

#### `at($content)`
@某人。`$content` — QQ 号，填 `all` 为全体
```php
群(群号, at(QQ), text(" 你好"));
```

#### `kp($content)`
JSON 卡片消息段。`$content` — JSON 字符串
```php
群(群号, kp($jsonStr));
```

### 消息发送

#### `群($group, ...$msgs)`

发送群消息。`$group` — 群号；`...$msgs` — 消息段
```php
群(群号, text("Hello"), image("http://..."));
```

#### `私($group, ...$msgs)`
发送私聊消息。`$group` — 对方 QQ；`...$msgs` — 消息段
```php
私(QQ, text("私信内容"));
```

#### `伪造($QQ, $name, ...$msgs)`
构造合并转发节点。`$QQ` — 伪造QQ；`$name` — 伪造昵称；`...$msgs` — 消息段
```php
$node = 伪造(10001, "小明", text("伪造消息"));
```

#### `群伪造($group, ...$msgs)`
向群发合并转发。`$group` — 群号；`...$msgs` — `伪造()` 节点
```php
群伪造(群号, 伪造(10001,"A",text("第一条")), 伪造(10002,"B",text("第二条")));
```

#### `私伪造($group, ...$msgs)`
向好友发合并转发。`$group` — 对方 QQ；`...$msgs` — `伪造()` 节点
```php
私伪造(QQ, 伪造(10001,"A",text("内容")));
```

### 消息操作

#### `取消息($raw, $type)`
从消息数组中提取指定类型内容。`$raw` — 原始消息数组；`$type` — `reply`/`json`/`audio`/`video`/`at`/`image`。`at` 和 `image` 返回数组，其余返回字符串。
```php
$images = 取消息(raw, "image");
$reply_id = 取消息(raw, "reply");
```

#### `获取消息($id)`
通过消息 ID 获取消息详情。`$id` — 消息 ID
```php
$data = 获取消息(消息ID);
```

#### `撤回消息($id)`
撤回指定消息。`$id` — 消息 ID
```php
撤回消息(消息ID);
```

#### `CQ_message($raw)`
CQ 码转可读文本（图片→`[图片]`，@→`[艾特]` 等）。`$raw` — 原始消息字符串
```php
$msg = CQ_message(raw["raw_message"]);
```

### 群管理

#### `禁言($group, $user, $time)`
禁言成员。`$group` — 群号；`$user` — QQ；`$time` — 秒（0为解禁）
```php
禁言(群号, QQ, 600);
```

#### `全体禁言($group)`
开启全体禁言。`$group` — 群号
```php
全体禁言(群号);
```

#### `踢($group, $user, $back)`
踢出成员。`$group` — 群号；`$user` — QQ；`$back` — 是否拒绝再加，默认 `false`
```php
踢(群号, QQ);
踢(群号, QQ, true);
```

#### `设置群管理员($group, $user, $back)`
设置/取消管理员。`$back` — `true` 设置，`false` 取消，默认 `true`
```php
设置群管理员(群号, QQ);
设置群管理员(群号, QQ, false);
```

#### `设置群名称($group, $name)`
修改群名称。`$name` — 新群名
```php
设置群名称(群号, "新名字");
```

#### `设置群名片($group, $user, $name)`

修改成员群名片。`$name` — 名片内容
```php
设置群名片(群号, QQ, "靓仔");
```

#### `设置头衔($group, $user, $title)`
设置专属头衔。`$title` — 头衔文字
```php
设置头衔(群号, QQ, "最强王者");
```

#### `退群($group, $is_dismiss)`
机器人退群。`$is_dismiss` — 是否解散群，默认 `false`
```php
退群(群号);
```

#### `群打卡($group)`
群签到。
```php
群打卡(群号);
```

#### `群戳一戳($group, $qq)`
群内戳一戳。`$qq` — 目标 QQ
```php
群戳一戳(群号, QQ);
```

#### `私戳一戳($qq)`
私聊戳一戳。
```php
私戳一戳(QQ);
```
