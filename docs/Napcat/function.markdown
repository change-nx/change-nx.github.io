### BOTAPI()
> BOTAPI($url,$json)

底层API调用函数。

| 参数名 | 类型 | 介绍 |
|-------|------|------|
| $url | string | API路径（不带基础URL） |
| $json | mixed | 请求参数（数组或0） |

### text()
> text($content)

构建文本消息段。

| 参数名 | 类型 | 介绍 |
|-------|------|------|
| $content | string | 文本内容 |

使用示例:
```php

```

### 伪造()
> 伪造($QQ,$name,...$msgs)

构建合并转发消息节点。

| 参数名 | 类型 | 介绍 |
|-------|------|------|
| $QQ | int/string | 伪造的发送者QQ号 |
| $name | string | 伪造的发送者昵称 |
| ...$msgs | mixed | 可变参数，一个或多个消息段 |

使用示例:
```php

```

> CQ_message($raw)

## 消息发送函数

> 群($group,...$msgs)
