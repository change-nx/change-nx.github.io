一、通用函数

1. curl() - HTTP请求

```php
curl("https://api.example.com", "POST", ["Content-Type: application/json"], '{"data":"test"}')
```

功能：发送HTTP请求
参数：URL, 请求方式(GET/POST/PUT/DELETE), 请求头数组, 请求体
返回：响应内容

2. 读() - 读取配置

```php
读("config.json", "api_key", "")
```

功能：从文件读取配置值
参数：文件路径, 键名, 默认值
返回：对应的值或默认值

3. 写() - 写入配置

```php
写("config.json", "api_key", "123456")
```

功能：写入配置到文件
参数：文件路径, 键名, 值

4. 二维码() - 生成二维码

```php
二维码("https://qq.com")
```

功能：生成二维码图片
参数：文本或URL
返回：Base64编码的图片字符串

5. load() - 加载脚本

```php
load(__DIR__."/plugins")
```

功能：导入指定目录所有PHP脚本
参数：目录路径

二、消息构造函数

6. text() - 文本消息

```php
text("欢迎使用机器人")
```

功能：构造纯文本消息段

7. reply() - 回复消息

```php
reply(123456789)
```

功能：回复指定消息ID的消息

8. face() - QQ表情

```php
face(178) // 微笑表情
```

功能：构造QQ表情
参数：表情ID(1-221)

9. image() - 图片消息

```php
image("https://example.com/img.jpg")
image("data:image/png;base64,iVBORw0...")
```

功能：构造图片消息
参数：URL或Base64编码

10. video() - 视频消息

```php
video("https://example.com/video.mp4")
```

功能：构造视频消息

11. music() - 音乐分享

```php
music("qq", "0039MVlR4Kz4Lq") // QQ音乐
music("163", "1498320418")   // 网易云音乐
```

功能：构造音乐卡片
类型：qq(QQ音乐)、163(网易云)、xiami(虾米)

12. at() - @某人

```php
at(10001)
```

功能：@指定群成员

13. kp() - JSON卡片

```php
kp('{"app":"com.tencent.miniapp","desc":"小程序"}')
```

功能：发送JSON格式卡片消息

14. 伪造() - 合并转发节点

```php
伪造(10001, "张三", text("你好"), image("url"))
```

功能：构造合并转发的一个节点
参数：QQ号, 昵称, 消息段...

三、消息发送函数

15. 群() - 发送群消息

```php
群(123456, text("群公告"), at(10001), image("url"))
```

功能：向指定群发送消息

16. 私() - 发送私聊

```php
私(10001, text("你好"), music("qq", "001"))
```

功能：向好友发送私聊消息

17. 群伪造() - 群合并转发

```php
$n1 = 伪造(10001, "A", text("第一句"))
$n2 = 伪造(10002, "B", text("第二句"))
群伪造(123456, $n1, $n2)
```

功能：发送群合并转发消息

18. 私伪造() - 私聊合并转发

```php
私伪造(10001, $n1, $n2)
```

功能：发送私聊合并转发消息

四、消息操作函数

19. 取消息() - 提取消息内容

```php

取消息($raw, "json")    // 返回JSON内容
```

功能：从原始消息提取特定内容
