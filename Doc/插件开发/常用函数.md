### 工具函数

#### `wlog($name, $content)`
写入日志文件，按日期自动分文件存储在 `Log/{name}/` 目录下。`$name` — 日志分类名；`$content` — 日志内容
```php
wlog(adapter, "发生了某件事");
wlog("debug", json_encode($data));
```

---

#### `curl($url, $method, $headers, $params)`
发送 HTTP 请求。`$url` — 请求地址；`$method` — `GET`/`POST`/`PUT`/`DELETE`；`$headers` — 请求头数组；`$params` — 请求体（数组或字符串）
```php
$res = curl("https://example.com/api", "POST", ["Content-Type: application/json"], $jsonStr);
```

---

#### `前缀($str, $prefix)`
判断字符串是否以指定前缀开头，返回 `true`/`false`。`$str` — 原始字符串；`$prefix` — 前缀
```php
if (前缀(消息, "!点歌")) { ... }
```

---

#### `前缀后($str, $prefix)`
去掉字符串开头的前缀并返回剩余内容，不匹配则原样返回。`$str` — 原始字符串；`$prefix` — 前缀
```php
$keyword = 前缀后(消息, "!点歌 "); // "!点歌 周杰伦" → "周杰伦"
```

---

#### `写($文件, $键, $值)`
向 `database/` 目录下的 JSON 文件写入键值，文件不存在自动创建，返回 `true`/`false`。`$文件` — 文件路径（相对 database/）；`$键` — 键名；`$值` — 值

```php
写("user/data.json", QQ, 100);
```

---

#### `读($文件, $键, $默认值)`
从 `database/` 目录下的 JSON 文件读取键值，文件或键不存在返回默认值。`$默认值` 默认 `null`
```php
$score = 读("user/data.json", QQ, 0);
```

---

#### `二维码($content)`
生成二维码并返回 base64 字符串（可直接用于图片发送）。`$content` — 二维码内容
```php
群(群号, image("base64://" . 二维码("https://example.com")));
```

---

#### `puppeteer($html, $data)`
调用本地 Puppeteer 插件对 HTML 文件截图，返回图片 base64，失败返回 `null`。需在 NapCat 插件商店安装 puppeteer 插件。`$html` — HTML 文件路径；`$data` — 数组，含 `data`（模板变量）、`width`、`height`
```php
$img = puppeteer("template/card.html", [
    "data" => ["name" => "张三", "score" => 99],
    "width" => 600,
    "height" => 400
]);
if ($img) 群(群号, image("base64://" . $img));
```
